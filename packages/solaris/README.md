# Solaris packages

In this repository, you can find the necessary tools to build a ShieldnetDefend package for solaris 10 and 11, using scripts or automated using vagrant, and the tools to build a solaris 10 and 11 vagrant box:

- [Automated Solaris package generation using vagrant](/solaris/package_generation/README.md)
- [Vagrantbox using packer](/solaris/packer/README.md)

## Building Solaris packages

Please, visit the following link for the full Solaris packages building documentation: [Generate ShieldnetDefend Solaris packages.](https://documentation.shieldnetdefend.com/current/development/packaging/generate-sol-package.html)

ShieldnetDefend RPM packages for IBM AIX
==============================

In this repository, you can find the necessary tools to build a ShieldnetDefend package for AIX.

## Building AIX packages

Please, visit the following link for the full AIX packages building documentation: [Generate ShieldnetDefend AIX packages.](https://documentation.shieldnetdefend.com/current/development/packaging/generate-aix-package.html)
